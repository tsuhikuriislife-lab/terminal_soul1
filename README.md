# Terminal Souls

## Description

A turn based fighting game where you control the actions of a hero against an enemy.

## How to install the game

- First, you need a Visual Studio Code or any other Code Editor to run this script.
- After that, you download the main script called "terminal.py"
- Use the command "python3 terminal.py" to play!

## Features

- Turn based combat
- Different types of actions for the hero like attack, heal and use a skill!
- The oponent can heal himself when it gets low on health
- A range of difficulties to choose
- Some things are randomized, like the damage dealt and critical rate

## How to play the game

At the start, you select the difficulty and the program assigns your attributes.

Then, you can select what your hero is going to do.
- Attack: It picks a random number between a set range, it has a small chance to deal double damage
- Heal: If tou have enough potions, your Hero will heal a set amount of health
- Skill: It can deal a large amount of damage, but has a chance to miss and do nothing. If it hits, it has a small chance to deal double damage

After you use your turn, the enemy attacks you.

- It will deal a random amount of damage in a set range, it has a small chance to deal double damage
- If the enemy gets low on health, it will recover a set amount of health

#

If you defeat the enemy, you win!
If the enemy defeats you, you lose!
